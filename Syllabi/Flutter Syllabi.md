## Silabus Pemrograman Flutter

### **Pendahuluan**

- **Apa itu Flutter?**
  - Sejarah singkat dan perkembangan Flutter
  - Keunggulan Flutter dibandingkan framework lain (cross-platform, hot reload, UI yang indah)
  - Penggunaan Flutter di industri
- **Dart: Bahasa Pemrograman untuk Flutter**
  - Sintaks dasar Dart (variabel, tipe data, operator, kontrol aliran)
  - Konsep OOP dalam Dart (kelas, objek, inheritance, polymorphism)
  - Fitur-fitur Dart yang penting untuk Flutter (async/await, generics)

### **Mulai Mengembangkan dengan Flutter**

- **Set up lingkungan pengembangan:**
  - Instalasi Flutter SDK
  - Pemilihan editor kode (VS Code, Android Studio, IntelliJ)
  - Membuat project Flutter pertama
- **Widget Dasar:**
  - Konsep widget (StatelessWidget, StatefulWidget)
  - Widget umum (Text, Container, Row, Column, Image, Button)
  - Layout (Stack, GridView, ListView)
  - Styling (ThemeData, Color, Font, BoxDecoration)
- **Navigasi:**
  - Named routes
  - MaterialPageRoute
  - Passing arguments antara layar
- **State Management:**
  - **Provider:** Konsep provider, Consumer, SelectProvider
  - **Riverpod:** Konsep provider, consumer, state notifier
  - **Bloc:** Konsep bloc, event, state, BlocProvider
  - Memilih state management yang tepat untuk proyek

### **Interaksi Pengguna**

- **Gesture detector:** Tap, double tap, long press, drag
- **Form:** TextField, TextFormField, validation
- **SnackBar**
- **Dialog**

### **HTTP Requests**

- **http package:** GET, POST, PUT, DELETE
- **Parsing JSON**
- **Menampilkan data dari API**

### **Platform Channels**

- **Komunikasi dengan native code (Android/iOS)**
- **Plugin Flutter**

### **UI yang Lebih Menarik**

- **Custom paint**
- **Animation**
- **Theme customization**

### **Testing**

- **Unit testing**
- **Widget testing**
- **Integration testing**

### **Deployment**

- **Menerbitkan aplikasi ke App Store dan Google Play Store**
- **Build untuk web**
